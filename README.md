## 功能特性

- 深色主题
- 本地化界面语言
- 首页置顶文章
- 分层分类目录
- 热门标签
- 目录（TOC）
- 最后修改时间
- 代码语法高亮
- 数学公式渲染
- Mermaid 图表与流程图
- 深色模式图片适配
- 媒体嵌入
- 评论系统
- 内置搜索
- Atom 订阅
- PWA 支持
- 网站访问分析
- SEO 与性能优化
